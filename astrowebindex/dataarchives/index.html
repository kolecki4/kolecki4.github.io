Data Archives By Telescope

Calar Alto Observatory (CAHA)
For raw and reduced data from Calar Alto
Canada-France-Hawaii Telescope (CFHT)
Contains raw and reduced data from CFHT, as well as a number of other observatories, including Hubble, Webb, and Gemini.
ESO Phase 3 Archive
For reduced data from the European Southern Observatory.
ESO Main Data Archive
Contains a wider range of instruments, but features mainly raw, unreduced data.
Data reduction tools FAQ page
Gemini Observatory Archive
For raw and reduced data products from the Gemini Observatory
Gran Telescopia Canarias (GTC) Archive
For raw and reduced data products of the GTC
Italian Center for Astronomical Archives
Some data archives listed here (LBT, TNG) are run by IA2, as well as others, which are listed at the bottom of their main page.
Infrared Science Archive (NASA/IPAC)
Contains data from numerous telescopes, including but not limited to WISE, 2MASS, Spitzer, and the Planck Telescope
Keck Observatory Archive (KOA)
For raw and reduced data from the Keck Telescope. Reduced data must be explicitly requested in the interface above the file list, otherwise only the raw images will be available
Data reduction tools
Large Binocular Telescope Archive
Raw data from the Large Binocular Telescope
Data reduction tools LBT Interferometer Archive
NOIRLab Astro Data Archive
Contains data from Kitt Peak, Bok, Cerro Tololo, SOAR, and WIYN Observatories
NEID Archive LBT Interferometer Archive
Subaru Telescope Archive (SMOKA)
Contains only raw data
Telescopio Nazionale Galileo Archive
Note that reduced products are interspersed with raw data, with no explicit flag to tell the difference. For example, in GIANO-B spectra, the reduced, 1D spectrum files have names which end in "ms1d"
